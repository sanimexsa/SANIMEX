# SANIMEX Agent Skills

This project adopts the [Anthropic Skills](https://github.com/anthropics/skills) architecture to standardize agent behavior, technical workflows, and quality engineering.

## üöÄ Technical Skills

### 1. [Webapp Testing](skills/webapp-testing/SKILL.md)
Guidelines for maintaining high-quality code via Vitest unit tests and Playwright E2E matrix testing across devices and languages.

### 2. [Git Workflow](skills/git-workflow/SKILL.md)
Mandatory procedure for bug fixing and feature development involving GitHub Issues, branching, and Pull Requests.

### 3. [Frontend Design](skills/frontend-design/SKILL.md)

Design system and aesthetic guidelines for creating distinctive, production-grade interfaces that avoid generic "AI slop" aesthetics. Defines typography, color palette, motion, and spatial composition standards.

### 4. [Web Artifacts Builder](skills/web-artifacts-builder/SKILL.md)

Patterns and standards for building React + TypeScript + Tailwind artifacts. Covers project structure, component templates, animation classes, i18n support, and testing requirements for the SANIMEX stack.

### 5. [Brand Guidelines](skills/brand-guidelines/SKILL.md)

SANIMEX S.A. brand identity codified as "Desert Modernism." Defines the complete color palette, typography rules, logo usage, spacing conventions, and visual treatment standards for consistent brand representation.
